## 🚀 Main Project

### I-Poten — AI 모의면접 및 학습 플랫폼

> IT 개발자 취업 준비생과 주니어 개발자를 위한 학습 서비스
> AI 모의면접 · IT 용어 학습 · 개인 단어장 · 퀴즈 기능 제공

| 기능 | 설명 |
|---|---|
| 🔍 **PotenWord** | IT 개발 용어를 분야별로 탐색·학습하는 용어 검색 서비스 |
| 📓 **PotenNote** | 필요한 용어를 저장하고 복습하는 개인 학습 노트 |
| ✏️ **PotenQuiz** | 저장 개념 기반의 반복 학습 퀴즈 기능 |

**담당 역할**
- React · TypeScript 기반 화면 구현 및 사용자 상태별 분기 처리
- Spring Boot · JPA 기반 API 개발, 예외 처리, 서비스 정책 반영
- 학습 흐름에 맞춘 UI/UX 개선 및 MySQL 조회 구조 최적화
