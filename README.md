## Project

### I-Poten — AI 모의면접 및 학습 플랫폼



IT 개발자 취업 준비생을 위한 서비스로, AI 모의면접 · 용어 학습 · 개인 단어장 · 퀴즈 기능을 제공합니다.

| 기능 | 설명 |
|---|---|
| 🔍 PotenWord | IT 용어를 분야별로 탐색·학습하는 검색 서비스 |
| 📓 PotenNote | 필요한 용어를 저장하고 복습하는 개인 학습 노트 |
| ✏️ PotenQuiz | 저장 개념 기반의 반복 학습 퀴즈 |

**역할** React · TypeScript 화면 구현 / Spring Boot · JPA API 개발 / UI·UX 개선 / MySQL 조회 구조 최적화

🌐 [i-poten.com](https://i-poten.com) · 🔗 [Backend](https://github.com/Mur-pixel/I-Poten-Backend) · 🔗 [Frontend](https://github.com/Mur-pixel/I-Poten-Frontend)
